# Desercion-de-estudiantes-de-educacion-superior
La deserción académica en la educación superior representa un problema que afecta el desarrollo social y económico del país. Se relaciona con la dificultad de identificar a tiempo los factores que llevan a los estudiantes a abandonar sus estudios, los cuales pueden ser socioeconómicos, académicos y demográficos, lo que requiere análisis de datos.
Análisis de Deserción Estudiantil (SPADIES) con Big Data

Este proyecto utiliza una arquitectura distribuida para procesar tasas de deserción y graduación del Ministerio de Educación Nacional de Colombia.

🛠️ Tecnologías Utilizadas
- Apache Kafka: Gestión de mensajes y flujo de datos en tiempo real.
- Apache Spark Streaming: Procesamiento y analítica distribuida.
- Python: Ingesta y transformación de datos.

📊 Datos Utilizados
Los datos provienen del sistema SPADIES, enfocándose en las métricas de las IES (Instituciones de Educación Superior) entre 2010 y 2023.
